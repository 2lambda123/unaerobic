02-27 23:34:30.108: DEBUG/zzzPostResponce(30229): <!>com.kovalenych.RankingActivity 217<!>   <p>APNEA.cz Ranking was built to help freedivers getting better, and more complete overview and analysis of their performances. It contains not only data from the official <a href="http://www.aidaregister.org">AIDA Result Register</a>, but also from many other sources: <a href="http://freedivecentral.com/" target="_blank">Freedive Central</a>, <a href="http://apneamania.com/" target="_blank">Apnea Mania</a>, <a href="http://apnee.ffessm.fr/" target="_blank">FFESSM</a>, <a href="http://www.cmas2000.org/" target="_blank">CMAS</a>, <a href="http://portale.fipsas.it/" target="_blank">FIPSAS</a>, <a href="http://www.divingfree.org/" target="_blank">F.R.E.E.</a>, and also directly from competition organizers. Data from external sources was not blindly imported, but carefully analyzed, and many duplicate entries, misspellings, false name assignments, and other discrepancies were removed.
